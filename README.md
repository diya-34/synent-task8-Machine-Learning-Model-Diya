# synent-task8-Machine-Learning-Model-Diya

🏦 AI Powered Loan Prediction & Banking Analytics System

An AI-powered Loan Prediction System developed using Machine Learning, Streamlit, and Power BI.
This project predicts whether a loan application will be Approved or Rejected based on customer financial information and provides interactive banking analytics.

🚀 Project Overview

The main objective of this project is to automate the loan approval prediction process using Machine Learning techniques.

The system analyzes customer information such as:

Applicant Income
Coapplicant Income
Credit History
Education
Employment Status
Loan Amount
Property Area

and predicts the loan approval result.

✨ Features
✅ Loan Prediction System

Predicts loan approval status using a trained Machine Learning model.

📊 Banking Dashboard

Professional banking-style dashboard with KPIs and analytics.

💳 EMI Calculator

Calculates estimated monthly EMI based on loan amount and interest rate.

📈 Customer Credit Score

Displays estimated customer credit score and profile analysis.

📄 PDF Report Generation

Automatically generates downloadable PDF reports.

📧 Email Notification

Sends loan prediction report through email.

🤖 AI Loan Assistant

Simple AI assistant for loan-related queries.

🎤 Voice Assistant

Voice assistant simulation feature.

🗄️ Prediction History

Stores prediction records using SQLite database.

📊 Power BI Dashboard

Interactive financial analytics dashboard with charts and KPIs.
